# tsumiki-legal

Tsumiki (iOS アプリ) のプライバシーポリシー・利用規約・サポートページを GitHub Pages でホスティングするリポジトリ。

## ファイル

- `_config.yml` — Jekyll 設定 (theme: minima)
- `index.md` — ランディングページ
- `privacy-policy.md` — プライバシーポリシー
- `terms-of-service.md` — 利用規約
- `support.md` — サポートページ

## 公開 URL (GitHub Pages 有効化後)

- ランディング: https://jasmine-0101.github.io/tsumiki-legal/
- プライバシーポリシー: https://jasmine-0101.github.io/tsumiki-legal/privacy-policy.html
- 利用規約: https://jasmine-0101.github.io/tsumiki-legal/terms-of-service.html
- サポート: https://jasmine-0101.github.io/tsumiki-legal/support.html

## 編集と公開

ローカルで `.md` ファイルを編集 → `git push` するだけで GitHub Pages が自動更新する。

## ソース管理

本ファイルは Tsumiki Flutter プロジェクトの `docs/legal/` 以下にもコピーがある (アプリの内部ドキュメント参照用)。`<氏名>` `<contact email>` `<リリース日>` のプレースホルダはリリース直前に実値に差し替える。
